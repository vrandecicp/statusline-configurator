    1. `DEFAULT_ENABLED_IDS: string[]` = `['model_display_name']`

    2. `toggle(prev: string[], id: string): string[]`
       If id is in prev, return prev.filter(x => x !== id). Otherwise return [...prev, id].

    3. `moveUp(prev: string[], id: string): string[]`
       Find idx = prev.indexOf(id). If idx <= 0, return prev unchanged.
       Otherwise clone, swap [idx-1] and [idx], return clone.

    4. `moveDown(prev: string[], id: string): string[]`
       Find idx = prev.indexOf(id). If idx < 0 or idx >= prev.length - 1, return prev unchanged.
       Otherwise clone, swap [idx] and [idx+1], return clone.

    All functions must use immutable patterns (return new arrays, never mutate prev).
